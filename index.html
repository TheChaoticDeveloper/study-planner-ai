About This Project

This project is a web-based AI assistant designed to help students organize their studying and plan upcoming future events.

This AI can help create study schedules, explain study strategies, and assist with planning future assignments.

Creator

Created by Andrew Bachez (Me) as part of my IB Personal Project focused on creating an AI that helps with both studying and planning.

How To Use

  • Type a question in the chat box
  • Ask for help with studying or planning
  • Example: "Make a 5 day study plan for math"
  • Press Enter or The Send Button

The AI should respond with structured advice or planning suggestions.
